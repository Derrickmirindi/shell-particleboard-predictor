Shell-Based Particleboard Property Predictor

Random Forest Linear Equations — Predict TS, MOR, and IB from input properties

Thickness Swelling (TS)

TS = 0.0027 × Density + 0.2743 × WA − 2.3678
Typical ranges: Density 518–1585 kg/m³, WA 5–90%
—

Modulus of Rupture (MOR)

MOR = 0.0055 × Density + 0.0021 × MOE + 1.4443
Typical ranges: Density 518–1585 kg/m³, MOE 11–4190 MPa
—

Internal Bond Strength (IB)

IB = 0.0054 × Density + 0.0007 × MOE − 4.0890
Typical ranges: Density 518–1585 kg/m³, MOE 11–4190 MPa
—

This web application uses a trained Random Forest (RF) to predict the thickness swelling (TS), modulus of rupture (MOR) and internal bond strength.
It provides fast estimates of the shell-based particleboards' properties with TS at R² value of 0.83, MOR R² value of 0.852, and IB R² value of 0.75. Random Forest model predictions.
The model was trained on data defined over specific value ranges, so users should enter numerical values only within the indicated intervals for each input parameter.
